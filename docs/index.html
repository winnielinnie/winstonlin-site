Developer platforms, serverless, and infrastructure

Most of my work has been in CI/CD, Kubernetes, and serverless. I still like building small Python tools when it helps make something clearer or easier to use.

Most of my work sits somewhere between product, infrastructure, and developer experience. Over the last several years I have worked on CI/CD, Kubernetes, and serverless platforms inside OCI.

This is a simple place to keep the parts of that work that translate well in public: case studies, a few notes on platform work, and small Python projects I would actually want to maintain.

Some of the writing here first shows up on Oracle Blogs. Some of it starts as rough notes and gets cleaned up later. I would rather it be useful than polished.
